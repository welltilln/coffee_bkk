# coffee_bkk — BKK Merchandiser Routes + Coffee Shops Map

หน้าแผนที่ standalone แสดง route รายวันของ merchandiser ทุกพื้นที่ BKK (BKK01–BKK15) พร้อมกัน
ทับด้วยหมุดร้านกาแฟในกรุงเทพฯ 1,706 ร้าน (ข้อมูลจาก Longdo Map)

## วิธีใช้

เลือกอย่างใดอย่างหนึ่ง:

1. ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้ตรง ๆ (ไม่ต้องติดตั้งอะไร)
2. หรือรัน `run.bat` → เปิดเซิร์ฟเวอร์ที่ http://127.0.0.1:8790 (ต้องมี Python)

ต้องต่ออินเทอร์เน็ตเพื่อโหลดพื้นหลังแผนที่ (OpenStreetMap tiles) — ส่วนข้อมูล route และร้านกาแฟอยู่ในโฟลเดอร์ทั้งหมด

## ไฟล์

| ไฟล์ | คำอธิบาย |
|---|---|
| `index.html` | หน้าแผนที่ (Leaflet) — เส้น route สีตามวัน หมุดเลขลำดับร้าน หมุด ☕ กดดูรายละเอียดร้านกาแฟ |
| `routes_data.js` | ข้อมูล route 15 พื้นที่ BKK (เส้นทางบนถนน + จุดร้าน) |
| `coffee_data.js` | ข้อมูลร้านกาแฟ 1,706 ร้าน (ชื่อ/ที่อยู่/โทร/เว็บ/พิกัด) แปลงจาก `coffee_bkk.xlsx` |
| `coffee_bkk.xlsx` | ไฟล์ต้นทางข้อมูลร้านกาแฟ |
| `leaflet/` | ไลบรารี Leaflet 1.9.4 (vendored — ไม่พึ่ง CDN) |
| `run.bat` | เปิด local server พอร์ต 8790 |
